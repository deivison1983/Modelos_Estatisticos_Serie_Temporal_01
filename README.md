# 🌊 Análise de Série Temporal da Temperatura da Superfície do Mar (El Niño) com Modelagem Estatística

## 📌 Visão Geral

Este projeto realiza a análise estatística de uma série temporal referente à temperatura da superfície do mar, utilizando a base de dados El Niño disponível na biblioteca Statsmodels.

O foco principal é analisar o comportamento temporal da variável, identificar padrões, tendências e características estatísticas, além de aplicar técnicas de modelagem e diagnóstico estatístico apropriadas para dados temporais.

Este tipo de análise é fundamental em aplicações reais como previsão climática, modelagem econômica e sistemas de previsão baseados em séries temporais.

---

## 🎯 Objetivo

Aplicar técnicas de análise estatística e modelagem em séries temporais com os seguintes objetivos:

* Analisar o comportamento temporal da temperatura da superfície do mar
* Identificar padrões e características da série temporal
* Aplicar modelagem estatística utilizando regressão
* Avaliar a qualidade do modelo construído
* Validar os pressupostos estatísticos em contexto de séries temporais

---

## 🧠 Conceitos e Técnicas Aplicadas

Este projeto demonstra a aplicação de técnicas fundamentais de análise de séries temporais e estatística:

### Análise Exploratória de Séries Temporais

* Visualização da série temporal
* Identificação de tendências
* Identificação de padrões temporais
* Análise de comportamento ao longo do tempo
* Estatísticas descritivas da série

### Modelagem Estatística

* Regressão Linear aplicada a dados temporais
* Estimação de parâmetros via Ordinary Least Squares (OLS)
* Interpretação dos coeficientes do modelo

### Diagnóstico Estatístico do Modelo

* Análise de resíduos
* Avaliação da normalidade dos resíduos
* Verificação da homocedasticidade
* Avaliação da independência dos erros

### Análise de Dependência Temporal

* Identificação de autocorrelação
* Avaliação da independência das observações
* Análise do comportamento sequencial dos dados

## 📊 Etapas do Projeto

O projeto foi desenvolvido seguindo um pipeline estruturado de análise de séries temporais:

1. Carregamento do dataset El Niño utilizando Statsmodels
2. Inspeção inicial e validação da estrutura dos dados
3. Conversão e organização da variável temporal
4. Visualização da série temporal
5. Análise exploratória do comportamento da série
6. Construção do modelo de regressão
7. Avaliação estatística do modelo
8. Análise dos resíduos
9. Diagnóstico da qualidade e adequação do modelo
10. Interpretação dos resultados

## 📈 Resultados e Principais Insights

A análise da série temporal permitiu observar características importantes:

* A série apresenta comportamento temporal estruturado, com variações significativas ao longo do tempo.

* O modelo de regressão permitiu capturar parte do comportamento da série, demonstrando a aplicabilidade de técnicas estatísticas em dados temporais.

* A análise de resíduos evidenciou aspectos importantes relacionados à dependência temporal, destacando a importância de técnicas específicas para séries temporais.

* O projeto demonstra que dados sequenciais exigem validações adicionais em relação a modelos tradicionais de regressão.
